# AI_Travel_Planner
A generative AI system that creates personalized travel itineraries using large language models based on user preferences, trip duration, and destination.

## Problem Statement
Planning a trip can be time consuming and overwhelming, as travelers must search through multiple sources to find attractions, restaurants, and activities that match their interests. While large language models can generate travel recommendations, it is unclear how effectively they can produce structured, personalized itineraries that consider user preferences such as destination, trip duration, and interests.

This project explores how generative AI can assist with travel planning by automatically generating personalized travel itineraries.

## Proposed Method
This project will develop an AI-powered system that generates structured travel itineraries using a LLM.

The system will take user inputs such as:
- Destination city
- Trip duration (number of days)
- User interests (e.g., food, museums, nature)
- Budget level 

Using prompt engineering and an LLM API, the system will generate a day-by-day itinerary with recommended attractions, activities, and dining options.

The project will also explore how different prompting strategies or models affect the quality and personalization of the generated itineraries.

## Data Sources
The project will use publicly available travel information and datasets, including:
- Online travel datasets (e.g., Kaggle travel datasets)
- Public lists of tourist attractions and landmarks
- LLM-generated recommendations

These sources will be used to evaluate the relevance and realism of the generated itineraries.

## Tools & Technologies (Planned)
- Python
- Large Language Model APIs (e.g., OpenAI or OpenRouter)
- Prompt engineering techniques
- Possible simple interface (e.g., Streamlit)

## Expected Outcome
The final system will allow users to input travel preferences and receive a personalized itinerary generated by AI. The project will also analyze how well generative models can support automated travel planning.
